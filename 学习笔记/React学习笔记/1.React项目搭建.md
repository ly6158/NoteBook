### 以官方脚手架创建项目 
- [React官网](https://zh-hans.reactjs.org/)

```bash
# npx 是 npm 5.2+ 附带的 package 运行工具。
$ npx create-react-app ly-blog-web-react
# 进入目录
cd ly-blog-web-react
# 暴露出相关配置文件
npm run eject
# 启动项目
# npm start
```
